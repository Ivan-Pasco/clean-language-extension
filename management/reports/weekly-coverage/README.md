# Weekly Coverage Reports

This directory stores weekly spec coverage reports.

## Schedule
- **Frequency:** Weekly on Friday
- **Skill:** `/coverage`
- **Output:** `YYYY-WNN.md` format (year-week number)

## How to Run Manually
Use `/coverage` in a Claude session.

## Report Format
Each report contains:
- Grammar production coverage percentages
- Semantic rule coverage percentages
- Type system coverage percentages
- Uncovered areas by priority
- Suggested tests to write
