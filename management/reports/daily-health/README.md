# Daily Health Reports

This directory stores daily health check reports generated by the scheduled health check task.

## Schedule
- **Frequency:** Daily at 8:00 AM local time
- **Agent:** QA agent (`/.claude/agents/qa.md`)
- **Output:** `YYYY-MM-DD.md` format

## How to Run Manually
```bash
# From project root
bash scripts/run_ci_tests.sh --verbose
```

## Report Format
Each report contains:
- Unit test results (pass/fail counts)
- CI test results by tier
- Baseline comparison (regressions/improvements)
- Spot-check compilation results
- Marker count delta
