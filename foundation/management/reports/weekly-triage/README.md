# Weekly Triage Reports

This directory stores weekly cross-component prompt triage reports.

## Schedule
- **Frequency:** Weekly on Monday
- **Skill:** `/triage`
- **Output:** `YYYY-WNN.md` format (year-week number)

## How to Run Manually
Use `/triage` in a Claude session.

## Report Format
Each report contains:
- Pending cross-component prompts by target component
- Priority categorization (critical/high/medium/low)
- Stale prompt identification
- Recommended actions
